# Installers

Windows installer artifacts are generated by Tauri in:

- `packages/desktop/src-tauri/target/release/bundle/msi`

Build locally from repository root:

```powershell
./scripts/build-release-windows.ps1
```

CI/CD automation:

- Workflow: `.github/workflows/windows-release.yml`
- Publishes build artifacts on `main`
- Publishes GitHub Releases on tag pushes matching `v*`

Code signing secrets expected by the workflow:

- `WINDOWS_CERT_PFX_BASE64`
- `WINDOWS_CERT_PASSWORD`
- `WINDOWS_SIGN_TIMESTAMP_URL` (optional)
